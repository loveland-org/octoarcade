Technical Solution

Problem: The text "No pinned fields yet" was using font-weight: bold (700) making it appear too heavy compared to other interface elements.

Solution: Changed the font weight to font-weight: 500 (medium) to match the visual hierarchy used throughout the interface.

CSS Change Required:

/* BEFORE (problematic) */
.pinned-fields-empty-state {
    font-weight: bold; /* or font-weight: 700 */
}

/* AFTER (fixed) */
.pinned-fields-empty-state {
    font-weight: 500; /* Medium weight for better visual balance */
}

Implementation Notes:

  • The fix maintains readability while reducing visual weight
  • Font weight 500 (medium) provides better consistency with other UI elements
  • This change improves the overall visual hierarchy of the settings panel
  • The fix addresses the visual imbalance mentioned in issue #382
